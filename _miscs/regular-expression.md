- 去 HTML 标签：

- 汉字 中文 日文
	- [\u4e00-\u9fa5]

|      字符      |     等价      |    说明     | 用法 |
| :------------: | :-----------: | :---------: | ---- |
|  **限定字符**  |     ----      |    ----     |      |
|       ?        |    {0, 1}     | 0 <= ? <= 1 |      |
|       +        |     {1, }     |    >= 1     |      |
|       *        |     {0, }     |    >= 0     |      |
|      {n}       |               |   = n 次    |      |
|     {n, }      |               |   >= n 次   |      |
|     {n, m}     |               |             |      |
| **非打印字符** |     ----      |    ----     |      |
|       \S       | [^\f\n\r\t\v] |  非空白符   |      |
|       \s       | [\f\n\r\t\v]  |   空白符    |      |
|  **特殊字符**  |     ----      |    ----     |      |
|       ^        |               |    开始     |      |
|       $        |               |    结尾     |      |
|      [^]       |               | 非 [] 内容  |      |
|       \        |               |    转义     |      |
|       ()       |               |             |      |
|       []       |               |             |      |
|       {}       |               |             |      |
|       .        |               |  非换行符   |      |
|                |               |             |      |
|                |               |             |      |
|                |               |             |      |
|                |               |             |      |
|                |               |             |      |
